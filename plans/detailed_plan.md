# 鲲鹏920优化详细计划

## 一、评分标准分析

### 1.1 初赛评分公式
```
总分 = PassingRate × 40% + 性能得分 × 60%
```

### 1.2 性能得分计算
```
性能得分 = 100 × (0.6×η1 + 0.3×η2 + 0.1×η3) / m
```
- η1 = 最大加速比 (T0/T_min)
- η2 = 平均加速比 (T0/T_avg)  
- η3 = 最小加速比 (T0/T_max)
- m = 鲲鹏920线程数量 (192核)

### 1.3 加速比计算
```
η = T0 / T
```
- T0 = 单核执行时间
- T = 并行执行时间
